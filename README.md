# 🚀 100-Day Java Project Roadmap

Welcome! This repository tracks my journey through a rigorous, self-imposed 100-day coding challenge. Having mastered Java fundamentals (OOP, syntax, basic data structures, and exception handling), I am cementing my knowledge through pure, hands-on engineering. 

I am building **exactly one distinct Java project every single day** for 100 consecutive days.
Difficulty Split: 
   - 25 Easy projects (Days 1 to 25) - focusing on basic logic, CLI apps, and foundational syntax.
   - 50 Medium projects (Days 26 to 75) - focusing on OOP design patterns, file I/O, collections, multithreading/concurrency, and basic APIs.
   - 25 Hard projects (Days 76 to 100) - focusing on complex systems, full-stack integrations, advanced concurrency, performance optimization, and networking.

*Difficulty: Easy*

* **Day 1: Simple Command-Line Calculator**
* **Description:** A CLI application that takes two numbers and an operator from the user to perform basic arithmetic.
* **Concepts/Libraries:** `Scanner`, basic operators, switch-case blocks.
**Status**: Done

* **Day 2: Number Guessing Game**
* **Description:** The system generates a random number, and the user has to guess it with higher/lower hints.
* **Concepts/Libraries:** `java.util.Random`, `while` loops, conditional logic.
**Status**: Done

* **Day 3: Temperature Converter**
* **Description:** Converts temperatures between Celsius, Fahrenheit, and Kelvin based on user choice.
* **Concepts/Libraries:** Methods, basic data types (`double`), input validation.
**Status**: Done
